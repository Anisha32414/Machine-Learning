# Machine Learning Algorithms

## Overview

This repository contains practical implementations of fundamental Machine Learning algorithms using Python and Jupyter Notebooks. Each notebook demonstrates data preprocessing, model training, evaluation, and validation using appropriate performance metrics.

## Algorithms Implemented

* **Linear Regression**

* **Ridge Regression**

* **Lasso Regression**

  * Train-Test Split
  * Cross Validation (`cross_val_score`)
  * Evaluation using **R² Score** and **Adjusted R² Score**

* **Logistic Regression**

  * Train-Test Split
  * Evaluation using **Accuracy**, **Precision**, **Recall**, and **Confusion Matrix**

* **Decision Tree Classification**

  * Implemented using `DecisionTreeClassifier`

* **K-Means Clustering**

  * Optimal value of **K** determined using the **Elbow Method (WCSS)**
  * Validated using the **Silhouette Score**

## Tech Stack

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
